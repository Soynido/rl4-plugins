# RL4: Manage API Keys

Open the RL4 dashboard to manage your API keys and Personal Sync settings.

## Action

1. Print: "Opening RL4 key management dashboard..."
2. Open the dashboard using Bash:

```bash
open "https://rl4.ai/dashboard/api-keys" 2>/dev/null || xdg-open "https://rl4.ai/dashboard/api-keys" 2>/dev/null || echo "Visit: https://rl4.ai/dashboard/api-keys"
```

3. Print:
```
 ┌─ API KEY MANAGEMENT ────────────────────────────────────────────────┐
 │                                                                      │
 │  Your browser should open the RL4 dashboard.                         │
 │                                                                      │
 │  From there you can:                                                 │
 │  • View and revoke API keys                                          │
 │  • Set up Personal Sync for cross-device context                     │
 │  • Generate new keys for additional devices                          │
 │                                                                      │
 │  URL: https://rl4.ai/dashboard/api-keys                             │
 │                                                                      │
 └──────────────────────────────────────────────────────────────────────┘
```
